# 🌿 生態系ピラミッド ギャラリー

同じWi-Fiの端末から複数人でリアルタイム共有できる生態系ピラミッドアプリです。

## 必要なもの

- **Node.js**（v14以上）— https://nodejs.org からインストール
- 外部ライブラリ不要（npm install 不要）

## 起動方法

```bash
node server.js
```

起動すると以下のように表示されます：

```
🌿 生態系ピラミッド ギャラリーサーバー 起動
─────────────────────────────────────────────
📡 ローカル:   http://localhost:3000

同じWi-Fiの端末からは:
  http://[このPCのIPアドレス]:3000

IPアドレスの確認方法:
  Mac:     ifconfig | grep "inet "
  Windows: ipconfig
─────────────────────────────────────────────
```

## アクセス方法

| 端末 | URL |
|------|-----|
| サーバーPC本体 | http://localhost:3000 |
| 同じWi-Fiのスマホ・PC | http://192.168.x.x:3000 |

## 使い方

1. 全員がブラウザで上記URLを開く
2. 各自が名前を入力して「はじめる」
3. 「✏️ 編集」タブでピラミッドを調整（1秒後に自動保存）
4. 「🖼️ 一覧」タブで全員のピラミッドを確認
5. 変更はリアルタイムで全員に反映される

## 仕様

- 同時接続: 制限なし（5〜10人推奨）
- 通信: WebSocket（npm依存なし・Node.js標準ライブラリのみ）
- データ保存: メモリのみ（サーバー再起動でリセット）
- ポート: 3000（変更する場合は server.js の PORT を変更）
